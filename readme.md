# GrillMaps Berlin

An interactive map application showing verified grilling spots in Berlin for expats.

## Features

- Interactive map of verified Berlin grilling locations
- Mobile-friendly responsive design
- Detailed information about each location
- Navigation links to Google Maps
- Copy-to-clipboard functionality for addresses
- Expat-specific tips and accessibility information

## Live Demo

Visit the live application: [GrillMaps Berlin](https://grillmaps.vercel.app)

## Technologies Used

- Mapbox GL JS for interactive mapping
- Vanilla HTML, CSS, and JavaScript
- Deployed on Vercel

## Local Development

1. Clone this repository
2. Get a Mapbox access token from [Mapbox](https://account.mapbox.com/access-tokens/)
3. Replace the placeholder token in `index.html`
4. Open `index.html` in your browser
